Tout ce que vous voulez savoir sur CoproGest.

CoproGest est conçu pour tous les acteurs d'une copropriété : les syndics professionnels, les copropriétaires, les membres du conseil syndical et les locataires. Chaque profil dispose d'un accès adapté à ses besoins.
Le syndic crée une assemblée générale, définit les résolutions et invite les copropriétaires. Ces derniers reçoivent un lien sécurisé pour voter. Les résultats sont calculés automatiquement selon les tantièmes.
